## ğŸ’¡ How It Works

- User enters height (in cm) and weight (in kg).
- Clicks the **"Calculate BMI"** button.
- JavaScript calculates BMI using the formula:

  \[
  \text{BMI} = \frac{\text{Weight (kg)}}{(\text{Height (m)})^2}
  \]

- Displays the BMI value and health status.
